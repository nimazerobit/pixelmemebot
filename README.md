# Inline Meme Telegram Bot
Inline Telegram Meme bot with python and SQLite database.

## 🚀 Features
- Admin panel for managing bot and see bot daily and overall reports.
- Show recent used meme at top of inline results
- Convert mp3 and other audio formats to Telegram voice using ffmpeg.
- Force join and ban system for users.
- and etc...

## 📥 Installation & Setup
1.  Clone project:
    ```bash
    git clone https://github.com/nimazerobit/pixelmemebot.git
    cd pixelmemebot
    ```
2.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
3.  Install ffmpeg based on your OS:
	On Ubuntu:
    ```bash
    sudo apt install ffmpeg
    ```
5.  Modify config files:
    -   RENAME `config-example.json` to `config.json` and put your owns.
6.  Run the bot:
    ```bash
    python main.py
    ```
